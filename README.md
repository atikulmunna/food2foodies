# Food 2 Foodies - Console Food Order System

A C-based console application for ordering food and managing delivery personnel. 

## Features
- **User Authentication**: Secure Sign-up and Login.
- **Data Encryption**: Employee records are stored using XOR encryption.
- **Modular Architecture**: Code is split into logical modules for maintainability.
- **Interactive UI**: Uses ANSI escape codes for a professional terminal look.

## How to Run
1. Ensure you have GCC installed.
2. Run `make` in the terminal.
3. Start the app:
   - Windows: `food_system.exe`
   - Linux/Mac: `./food_system`
